
<h1>名词

<h2>意义<br>
名词一般都是记录、反映人、事物、现象的名称的词。

<h2>从语法功能上看

<h3>可以接受数量词短语修饰。

<h3>不能接受副词修饰
以下情况除外：<br>
第一，在固定格式中，名词可以受副词修饰。例如，人不人鬼不鬼。<br>
第二，在方位词、处所名词前，可以有副词，如：最上面，最左边等等。<br>

<h3>可以置于介词后面，构成介宾短语。

<h3>可以做主语、宾语，有时可以做定语

<h3>有些名词如时间、处所名词等可以直接做判断里的谓语。
例如，今天星期一。

<h3>其中时间，方位、处所类名词还可以直接修饰动词，做状语。
例如，我们明天去吧。

<h1>动词

<h2>从意义上看<br>
动词一般都是表示行为、动作或变化意义的词，因此，动词可以分为：

<h3>动作、变化动词：走、看、来、上、运输、命令、消失、出现和发展等。

<h3>心理动词：思念、喜爱、憎恨等。

<h3>能愿动词：应该、能够、可以、愿意等。

<h3>类比动词：像、似、如、仿佛等。

<h3>称述动词：姓、称、认为

<h3>形式动词：予以、加以、给以等等。

<h2>从功能上看

<h3>可以接受不同副词修饰<br>
判断、类比、能愿、称述等类动词多能接受“不”修饰，极少接受“没有”修饰，如不是（像、能、姓）“等；心理及部分类比动词可直接接受”很“或”非常“修饰，”“非常思念”等，其他类动词则没有这一能力。

<h3>可以作谓语或述语。

<h3>有些动词能带宾语，成为及物动词，有些不能带宾语，称为不及物动词。

<h3>带有趋向意味的动词，还可以置于其他动词后充当补语，表示前面动词行为的趋向，如“太阳爬上来”

<h3>能愿动词还可以置于其他动词前做状语，如“应该去”

<h3>形式动词只能带动动词做宾语，虽然它们没有任何动作意义，但却可以用行动前加强行为的处置意味，强化表达语气，多用于带有政府或团体指令性色彩的表述中。
“予以坚决打击”

<h2>从形态上看

<h3>行为类动词（包含心理动词）可以重叠，形式为“AA，AAB，ＡＢＡＢ。

<h3>很多动词后面可跟”着“，”了“、“过”等助词表示动作的时间状态，如“看着书”，“吃过饭”，“洗了衣服”。

<h1>形容词<br>

<h2>从意义上看<br>

形容词是用来表示人、事物的属性与状态的，因此，它分为：<br>
（1）、性质形容词，如：肥、软、酸、勇敢、美丽等。
（2）、状态形容词，如：雪白、漆黑、羞答答、黑不溜秋。

<h2>从功能上看

<h3>绝大多数形容词可以受“不”，“很”等副词修饰，但状态形容词中已包含极致程度意味的词不可以含有受程度副词修饰，如雪白，乌黑，碧绿等。

<h3>形容词可以作定语、状语、谓语。如伟大的历史，快说，景色美丽。

<h3>状语形容词作定语，一般要后附助词“的”。

<h2>从形态上看<br>
（1）、可以重叠，形式要有AA,AABB,ABAB。
（2）、重叠后在语音上重叠成分有些声调发生变化：如AA儿式的后面的A一般念阴平，如慢慢儿
（3）、重叠后的形容词有这样几个作用：表示程度加深、突出喜爱色彩、且有协调音节作用。

<h1>区别词

<h2>从意义上看<br>
区别词是表示事物属性并用来给事物分类的词。

<h2>从功能上看<br>
区别词的主要特征有：<br>
(1)、不受绝大多数副词修饰，可受否定副词“非”的修饰
（2）、主要做定语，少量的可以做状语，如“长期努力”、“局部完成。

<h2>从形态上看<br>
与形容词不同，区别词极少重叠。

<h1>数词

<h2>从意义上看<br>
数词是表示数目或顺次的词，根据其内部不同，可分为：
（1）、基数词：一、二、三……
（2）、序数词：第一、第二……
（3）、另外还有倍数、概数等类型。

<h2>从功能上看<br>
数词的语法特点有：
(1)、能够与量词组合成数量短语
（2）、经常以数量短语的形式做定语或补语。

<h1>量词<br>

<h2>从意义上看<br>
量词是涌来计量事物或动作单位的词，可分为两类：<br>
（1）、名量词：位、个、本、颗、条、把
（2）、动量词：趟、回、次、下、遍、阵

<h2>从功能上看<br>
量词的特点有：
（1）、一般情况下，它是数词与名词组合时必须要有的中介成分，而且在语义上量词与名词存在关联性。
（2)、名量词与数词组成数量短语，作定语或谓语，如一位先生。
（3）、动量词与数词组成数量短语、可以置于动词前做定语、状语。
（4）、重叠量词还可以做主语或状语，如个个都很棒。

<h2>从形态上看<br>
量词的特点是：大多数单音节量词可以重叠，而且形式和作用多样。如：
（1）、量词单独重叠，如AA 
（1）、与数词一起构成重叠形式：一A一A。

<h1>副词
<h2>从意义上看<br>
副词是表示程度、范围、时间等意义，对动作或属性进行限制、修饰的词。一般可分为几类：
（1）、表示程度：很、非常、十分、万分、极、格外、略微、更。
（2）、表示范围：都、总共、一起、仅仅、唯独、各自、一概等。
（3）、表示时间：曾经、已经、就、刚刚、立刻、将要、接着。
（4）、表示肯定：肯定、必定、确实、准保、一定等。
（5）、表示否定：不、没有、别、未必、勿、甭等。
（6）、表示情态：果然、忽然、居然、仍然、照样、幸亏、渐渐、特意等。
（7）、表示语气：或许、索性、难道、无非、竟然、简直、到底、偏等。

<h2>从功能上看<br>
副词的特点有：<br>
（1）、有些能够修饰动词、如程度、范围、时间、情态、语气副词等；有些还能修饰形容词，如程度、肯定、否定副词等。
（2）、绝大多数只可以做状语，只有少数几个程度副词还可以做补语。
（3）、副词一般置于主谓结构谓语或述语前，但也有少量语气副词还可以置于主谓结构前，如：难道他就不能来吗？
（4）、有些副词单用或连用还可以充当关联词，如：她一来了，我就走。
（5）、副词不互相修饰，但可以连用。不过，此时副词是分层次修饰谓词性中心成分的。

<h1>代词<br>
<h2>从意义上看<br>
代词是用来起称代词、指示或问询作用的词语。根据指称关系，代词可分为：<br>
（1）、人称代词：我们、你们、他们、她们、它们、大家、咱们、大伙儿、自己、别人等。
（2）、指示代词：这个、那个、本、各、某、另、其余、他等。
（3）、疑问代词：谁、什么、哪儿、怎样、几时、多少等。

<h2>从功能上看<br>
由于代词在使用时主要用来指代客观对象、行为或数量等，因词，从被指代成分的词性上看，有的是名词性，如：小王明天来，他会把材料带给你。<br>
有的是动词或形容词性的，如：你最近身体怎么样？他怎么你啦？<br>
具体来说，代词的语法特点表现为下面两点：
（1）、体词性代词一般不受其他词修饰，但近年来人称代词经常可以有定语修饰。如：生活困窘的他又回到故乡。
（2）、代词根据其指代的成分性质而充当不同的句法成分、可以做主语、宾语、定语；也可以作谓语，如你身体怎么样啦？可以做状语，如大家哥各说一句话。做补语如文章写得如何。
（3）、代词在具体语句中有两种指称类型：实指和虚指。

<h1>连词<br>
用来连接词、短语或句子的词叫连词。<br>
按照连词所连接的单位类型，连词可以分为：
（1）、主要连接词、短语的：和、同、跟、与、及、并、或、而等。
（2）、主要连接句子的：然而、但是、不仅、即使、如果、那么、虽然、但是。因为、既然、纵然等。

<h1>介词<br>
依附在实词前，用以引入时间、处所、方式、工具、对象等成分。
（1）、引入时间：自、打、从、于、在、当、随着、到等。
（2）、引入处所、方向：朝、向、往、沿着、顺着、从等。
（3）、引入对象、范围：比、与、同、跟、和、把、被、叫、连、除了、关于。
（4）、引入方式、手段：按、以、用、经过、通过、依照、根据等。
（5）、引入原因、目的：因为、由于、为了、为着等。

<h1>助词<br>
<h2>定义与分类
主要依附与实词或短语后边，用来表示结构关系结构或体等语法意义的词。根据语法作用，将助词分为：
（1）、表示结构：的、地、得、所、之、等。
（2）、表示动态：着、了、过。
（3）、表示比况：一般、一样、似得。
（4）、表示数量：来、把、们等。

<h1>语气词<br>
<h2>定义与分类<br>
依附于句末或句中，有成句作用，表示某种语气的词，称为语气词。<br>
分类：陈述句、疑问句、反问句、祈使句和感叹句。

<h1>叹词<br>
<h2>定义与分类<br>
分类：<br>
（1）、表示应答的：哎、啊、嗯、哦等。
（2）、表示情绪的：唉、咦、啊、哇、哼、呸、哎呀等。

<h1>拟声词<br>
<h2>定义与分类<br>
用来模拟各种动作、行为的声音的词叫拟声词。如：轰、砰、啪、哗啦、叮咚、滴答、轰隆隆、叽叽喳喳等等。
